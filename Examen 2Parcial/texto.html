Diseño Web

Definición

El diseño web es una actividad creativa enfocada en realizar la parte visual de entornos y materiales digitales. Se encarga de layouts e interfaces de plataformas digitales como un sitio web o una app y realiza una configuración visible y funcional
para el usuario al distribuir y conceptualizar diversos elementos. Un buen diseño web debe contar con varias características para que sea vistoso y funcional; la conjugación de estas características garantizarán el éxito del proyecto.
